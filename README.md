ELIAN PATRICIO TAPIA VILLCA 
ADOPTAPET es una app React creada con Vite para mostrar mascotas disponibles para adopción. Incluye componentes para listar mascotas, mostrar tarjetas de mascota y filtrar por especie.

Instrucciones para ejecutar localmente
-Abrir la terminal en adoptapet-react-2
-Instalar dependencias: npm install
-Iniciar el servidor de desarrollo: npm run dev
-Abrir la dirección que indique Vite en el navegador, normalmente: http://localhost:5173
